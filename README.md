# tech-interview-simulador
"Entrega do desafio 'IA Entrevistador'. Criei um prompt avançado no Google Gemini para simular uma entrevista técnica de alto nível. O projeto documenta a interação, onde demonstrei conhecimentos em Arquitetura de Software e resolução de problemas de escalabilidade, recebendo um feedback positivo e aprovação da IA."
